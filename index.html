Sinkronisasi ke GitHub & Backup
Fitur ini menyimpan semua data aplikasi (keuangan, inventaris, kategori, identitas, dll) ke sebuah file database.json di repository GitHub milik Anda. Dengan begitu, kalau Anda buka aplikasi ini dari laptop lain (lewat alamat GitHub Pages yang sama), data akan otomatis ditarik dan tetap sama — Anda hanya perlu mengingat/menyimpan Token-nya, karena nama repo otomatis terdeteksi dari alamat website ini. Token GitHub hanya tersimpan di browser Anda sendiri (localStorage), tidak pernah dikirim ke server lain selain ke GitHub langsung dari browser.
Kunci Aplikasi dengan Password (Lokal)
Kalau diaktifkan, setiap kali aplikasi ini dibuka (tab/browser baru) akan diminta password dulu sebelum bisa melihat data apapun. Ini kunci sederhana untuk mencegah orang iseng buka laptop, bukan pengaman tingkat tinggi — perlindungan yang sesungguhnya tetap dari repo GitHub yang Private.

Pengaturan Repository GitHub
✓ Terisi otomatis dari alamat GitHub Pages ini. Kalau sudah benar, cukup isi Token lalu klik Simpan Pengaturan — Anda tidak perlu ketik nama repo lagi di komputer manapun selama membuka lewat alamat GitHub Pages ini.
Tersimpan ✓
Kode Setup (untuk pindah ke laptop lain)
Daripada ketik ulang token yang panjang di laptop lain (rawan salah ketik), pakai cara ini: di laptop yang sudah diatur, klik Salin Kode Setup, simpan kode itu di password manager/notes aman. Di laptop baru, tempel kode itu ke kolom bawah lalu klik Terapkan Kode — Token, Repo, Branch, dan Path langsung terisi semua sekaligus.
⚠️ Kode ini berisi token asli (hanya di-encode, bukan dienkripsi) — perlakukan serahasia token itu sendiri.
Sinkronisasi Manual
"Simpan Data ke GitHub" = kirim data yang ada di laptop ini ke GitHub (menimpa isi database.json).
"Ambil Data dari GitHub" = tarik data terbaru dari GitHub ke laptop ini (menimpa data lokal, akan diminta konfirmasi).

Backup Manual ke File (di luar GitHub)
Simpan salinan seluruh data sebagai satu file .json ke komputer Anda. File ini bisa dipakai untuk mengembalikan data kapan saja, sebagai cadangan tambahan selain GitHub.
Batalkan Pengambilan Terakhir
Setiap kali Anda klik "Ambil Data dari GitHub" atau "Pulihkan dari File Backup", data yang tadinya tampil di laptop ini otomatis disimpan dulu di sini sebelum ditimpa. Kalau ternyata salah ambil, klik tombol di bawah untuk kembali ke kondisi sebelumnya.
